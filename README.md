### Cost Structure

### Pricing Tiers

- Masterpiece: $45-50 profit (75-83% margin)

### Viewing the Website
1. Open `index.html` in any modern web browser
2. No server or installation required - it's pure HTML/CSS/JavaScript

### Customization Guide

#### Update Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --coffee-dark: #3E2723;
```

1. Place images in `assets/images/`
2. Update image placeholders in `index.html`
3. Replace `.image-placeholder` divs with `<img>` tags

- All text is in `index.html`
- Update business information, testimonials, pricing
- Customize form fields and contact details
